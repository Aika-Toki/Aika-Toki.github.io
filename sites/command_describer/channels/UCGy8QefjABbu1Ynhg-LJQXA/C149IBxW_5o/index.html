    Bedrock Edition

    準備機構

    scoreboard objectives add timer1 dummy

    scoreboard objectives add timer2 dummy

    scoreboard objectives add timer3 dummy

    タイマー用のスコアボードをいじるコマンド

    scoreboard players add @a[tag=timer] timer1 5

    scoreboard players add @a[scores={timer1=100..}] timer2 1

    scoreboard players set @a[scores={timer1=100..}] timer1 0

    scoreboard players add @a[scores={timer2=60..}] timer3 1

    scoreboard players set @a[scores={timer2=60..}] timer2 0

    秒を刻む音(自由)

    execute @a[scores={timer1=0},tag=timer] ~ ~ ~ random.click @s ~ ~ ~ 1 0.3

    タイマーを開始&停止するコマンド(必須)

    execute @a ~ ~ ~ detect ~ ~-0.1 ~ iron_block 0 scoreboard players set @s timer1 0

    execute @a ~ ~ ~ detect ~ ~-0.1 ~ iron_block 0 scoreboard players set @s timer2 0

    execute @a ~ ~ ~ detect ~ ~-0.1 ~ iron_block 0 scoreboard players set @s timer3 0

    execute @a ~ ~ ~ detect ~ ~-0.1 ~ iron_block 0 tag @s add timer

    execute @a[tag=timer] ~ ~ ~ detect ~ ~-0.1 ~ obsidian 0 playsound random.levelup @s ~ ~ ~ 0.5 0.7
